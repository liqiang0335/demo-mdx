# 高级 MDXProvider 示例

这个示例展示如何使用 **MDXProvider** 提供多种自定义组件，包括覆盖默认的 HTML 元素和添加全新的组件。

## 自定义标题样式

在这个文档中，所有标题都有自定义样式。包括：
- 一级标题（蓝色带下划线）
- 二级标题（绿色）
- 三级标题（红色）
