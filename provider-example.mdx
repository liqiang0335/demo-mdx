# MDXProvider 示例

这是一个使用 MDXProvider 的示例。你会注意到所有的 **h1** 标题都是蓝色的，而 **h2** 标题是绿色的。

## 二级标题

使用 **MDXProvider** 可以让你全局定义组件，而不需要在每个 MDX 文件中单独导入它们。

<Highlight>
  这段文字使用了 Highlight 组件，由 MDXProvider 全局提供，无需导入。
</Highlight>

<Wrapper>
  ```js
  // 不需要在此 MDX 文件中导入 Wrapper 组件
  const text = "MDXProvider 使全局组件可用";
  console.log(text);
  ```
</Wrapper>
